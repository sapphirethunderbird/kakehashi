# kakehashi

Communicate Visually
This was originally designed for those who have difficulty with verbal communication. It allows users to communicate with pre-installed cards, as well as their own drawings. There is also an emergency contact section in cases of emergency with emergency contact information.

> Currently hosted on [Vercel](https://kakehashi-blue.vercel.app/)

## Building the Server

This app is powered by [Next.js](https://nextjs.org/docs)
`npx create-next-app@latest my-aac-app`

Opt for custom settings

1. TypeScript: `Yes`
2. Tailwind CSS: `Yes`
3. ESLint: `Yes`
4. App Router: `Yes`
5. Src Directory: `Yes`
6. Import Alias: `Yes` (Default @/\* is fine)

## Installing Dependencies

Inside of the `my-aac-app` directory, run this to install dependencies:
`npm install react-sketch-canvas lucide-react zustand idb-keyval clsx tailwind-merge`
